`:闘魂.挨拶`
これは歴とした[Elixir]コードです。

```elixir
defmodule :闘魂 do
  def 挨拶 do
    "元氣ですかーーーーッ！"
  end

  def 説明 do
    "「闘魂」とは己に打ち克ち、闘いを通じて自分の魂を磨くことである"
  end
end
```

```
:闘魂.挨拶

:闘魂.説明
```
